### Слайд 1: Вступительный крючок (1 мин)

**Тезисы для выступления:**

> "Поднимите руку, если вы использовали GitHub Copilot. Или ChatGPT для кода. Или вставляли stack trace в AI-чатбот с вопросом 'что не так?'
>
> Отлично. А теперь забудьте обо всём этом.
>
> Claude Code — это не автодополнение. Это не просто окно чата рядом с редактором. Это полноценный AI-агент, который может читать ваши файлы, выполнять команды, редактировать код и анализировать весь проект целиком. В этом курсе мы используем подход terminal-first, потому что он даёт наиболее чёткую ментальную модель, но тот же продукт уже доступен в IDE, на десктопе и в вебе. Это разница между тем, когда кто-то шепчет вам подсказки на ухо... и когда опытный инженер садится за ваш стол и программирует вместе с вами."

**Ключевой тезис**: Установите ожидание с самого начала — это агентная разработка, а не автодополнение.

### Слайд 2: Что такое Claude Code на самом деле? (2 мин)

**Тезисы для выступления:**

- Claude Code начинался как terminal-first инструмент от Anthropic — компании, стоящей за Claude
- Сегодня он доступен в терминале, IDE, на десктопе и в вебе; этот курс фокусируется на терминале, потому что так базовый рабочий процесс виден наиболее наглядно
- Под капотом у него есть **инструменты**: он может читать файлы (Read), искать (Grep), находить файлы (Glob), редактировать код (Edit) и выполнять команды Bash
- Думайте о нём как об AI-напарнике с доступом `sudo` к вашему проекту (с предохранителями)
- Он видит весь репозиторий — не только файл, который у вас открыт
- Он сохраняет контекст на протяжении всей беседы — вы можете обсуждать архитектуру, пошагово отлаживать баг или планировать целую фичу

**Аналогия**: "Копайлоты в IDE — это как автозамена на телефоне: полезно, но ограничено тем, что на экране. Claude Code — это как звонок коллеге, который может подключиться по SSH к вашей машине и реально посмотреть, что происходит."

**Интересный факт**: "Вот что удивительно — Claude Code был частично создан с помощью Claude Code. Инженеры Anthropic использовали ранние версии инструмента для разработки более поздних версий. Это как молоток, который помог выковать молоток лучше."

### Слайд 4: Проект курса — ai-coderrank (2 мин)

**Тезисы для выступления:**

- Покажите скриншот/демо ai-coderrank
- Это дашборд на Next.js 14, который сравнивает AI-модели для кодинга (GPT-4, Claude, Gemini и др.) по производительности и цене
- Стек технологий: React 18, TypeScript, Tailwind CSS, Recharts для визуализации
- Включает Docker multi-stage builds, K8s-манифесты, конфигурацию ArgoCD, GitHub Actions
- Почему этот проект? Потому что в нём есть всё — фронтенд, API-роуты, контейнеризация, оркестрация, CI/CD
- Он реалистичный, но не перегруженный. Такой проект можно запустить в стартапе.

> "Мы выбрали этот проект, потому что он затрагивает все уровни современного веб-приложения. Фронтенд-компоненты, API-роуты, Docker-сборки, Kubernetes-деплойменты, CI-пайплайны. Если Claude Code поможет вам разобраться в этом и вносить изменения — он поможет с чем угодно на работе."

### Слайд 5: Разбор затрат — Без сюрпризов (1 мин)

**Тезисы для выступления:**

- Подписка Anthropic Pro: $20/месяц — это даёт доступ к Claude Code
- Дроплет DigitalOcean (с блока 7): ~$24/месяц за дроплет с 4 ГБ RAM, на котором работает k3s
- Итого: примерно $44/месяц на время прохождения разделов по инфраструктуре
- Дроплет можно выключить сразу после курса
- Подписку Pro? Её вы оставите. Поверьте.

> "Я хочу быть прозрачным насчёт затрат. Это не бесплатный инструмент — но по цене он как подписка на Netflix, и он будет экономить вам часы каждую неделю. Это не маркетинг — это математика."

### Слайд 6: Что нужно перед стартом (1 мин)

**Тезисы для выступления:**

Короткий чеклист — убедитесь, что у всех есть:

- Терминал, в котором вам удобно работать (macOS Terminal, iTerm2, Warp, любой терминал Linux)
- Node.js 18 или новее (`node --version`) для работы с проектом после установки
- Установленный и настроенный Git (`git --version`)
- Аккаунт на GitHub (мы будем форкать проект)
- Аккаунт Anthropic с подпиской Pro — зарегистрируйтесь на [claude.ai](https://claude.ai)
- Используйте текущий официальный способ установки Claude Code: нативный установщик на macOS/Linux/WSL, Homebrew на macOS при желании, или WinGet на Windows

> "Если у вас ещё нет подписки Pro, сейчас самое время оформить. Она понадобится через три минуты, когда мы перейдём к практике."
