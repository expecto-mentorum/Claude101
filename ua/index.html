Що ви створите

Не демо-іграшка. Реальний Next.js dashboard-застосунок, контейнеризований, задеплоєний на k3s на DigitalOcean через GitOps, доступний з будь-якої точки інтернету.

До

ai-coderrank: лідерборд ШІ-моделей для кодингу.

Працює локально. Світла тема. Без інфраструктури.

npm run dev → localhost:3000

Після

Темна тема. Docker-образ. CI/CD пайплайн. k3s кластер. ArgoCD GitOps. Публічна IP-адреса.

http://<droplet-ip>:30080 → живий сайт в інтернеті

Claude Code використовувався для створення частин самого Claude Code. У цьому курсі ви використаєте Claude Code, щоб задеплоїти застосунок, який ранжує ШІ-моделі для кодингу — включно з Claude. Це ШІ на всіх рівнях.

Розробники

Чому Claude Code у 2026

Claude Code — це не просто автодоповнення. Це повноцінний ШІ-агент, який читає вашу кодову базу, виконує команди, редагує файли, керує git та оркеструє мультиагентні процеси. Ось чим ви оволодієте:

Плани та ADR

Claude думає перед тим, як діяти. Увімкніть plan mode, генеруйте Architecture Decision Records та створюйте Mermaid-діаграми — все з розмови.

Власні навички

Упакуйте командні процеси в повторно використовувані slash-команди: /review-k8s, /check-docker, /deploy. Ваш плейбук, на базі Claude.

Хуки та автоматизація

Автоформатування при збереженні, блокування небезпечних команд, сповіщення при завершенні. Хуки запускають shell-команди на кожному етапі життєвого циклу.

MCP-сервери

Підключіть Claude до GitHub, Slack, Linear, баз даних та 50+ зовнішніх інструментів через Model Context Protocol.

Суб-агенти та команди

Створюйте спеціалізованих ШІ-працівників для перевірки безпеки, валідації K8s або паралельних досліджень. Координуйте мультиагентні команди для складних задач.

CI/CD та GitHub Actions

Позначте @claude в PR для автоматичного рев'ю. Створіть issue та дозвольте Claude реалізувати його. Плануйте регулярні хмарні задачі.

Курс

14 блоків. Кожен наступний спирається на попередній. Наприкінці ви задеплоїте реальний застосунок та опануєте Claude Code.

00

Вітання та налаштування

Встановіть Claude Code, автентифікуйтесь, зробіть форк проєкту та проведіть першу розмову з ШІ.

install /help
01

Розуміння вашої кодової бази

Дозвольте Claude проаналізувати проєкт, згенерувати CLAUDE.md та дослідити архітектуру.

/init CLAUDE.md
02

Запуск та тестування

Налаштуйте локально, запустіть тести, зберіть Docker — і подивіться, як Claude обробляє помилки.

Bash tool permissions
03

Планування з ADR та діаграмами

Думайте перед тим, як кодити. Створіть ADR, Mermaid-діаграми та план реалізації.

plan mode ADR Mermaid
04

Внесення змін — темна тема

Реалізуйте темну тему, перегляньте попередньо та закомітьте — Claude керує git-процесом.

Edit/Write git
05

Пам'ять та інтелект проєкту

Навчіть Claude ваших вподобань, правил команди та конвенцій проєкту — між сесіями.

.claude/rules /memory auto memory
06

Власні навички

Створіть повторно використовувані slash-команди для рев'ю K8s, аудиту Docker та командних процесів.

/skills $ARGUMENTS /simplify
07

Інфраструктура — k3s на DO

Створіть дроплет на DigitalOcean, встановіть k3s та задеплойте застосунок у Kubernetes.

k3s DigitalOcean
08

Хуки — автоматизація процесів

Автоформатування, сповіщення, блокування небезпечних команд — хуки спрацьовують на кожному етапі життєвого циклу.

hooks settings.json
09

MCP-сервери

Підключіть Claude до GitHub, файлових систем та зовнішніх API через Model Context Protocol.

MCP mcp.json
10

GitHub Actions та CI/CD

Автоматизуйте рев'ю PR, реалізацію issue та якість коду з Claude у CI.

@claude GitHub Actions
11

Суб-агенти

Створіть спеціалізованих ШІ-працівників для перевірки безпеки, валідації K8s та паралельних задач.

/agents worktree
12

GitOps-фінал — ArgoCD

Грандіозний фінал. Пушимо в git, ArgoCD синхронізує, темна тема з'являється в інтернеті.

ArgoCD /schedule MILESTONE
13

Просунуті патерни

Команди агентів, плагіни, headless-режим, Desktop-застосунок — та що далі.

teams /batch plugins

Що потрібно

Мінімум вимог, мінімум витрат. У цьому й суть.

Передумови

  • Ви знаєте git і вмієте відкрити термінал
  • Базове знайомство з концепціями Docker
  • Обліковий запис GitHub
  • Обліковий запис DigitalOcean (нові акаунти отримують $200 безкоштовного кредиту)
  • Цікавість та приблизно 10 годин зосередженого часу

Досвід з Kubernetes не потрібен — ми навчимо в процесі.

Витрати

Підписка Claude Pro $20/міс
Дроплет DigitalOcean (s-2vcpu-4gb) ~$24/міс
Доменне ім'я (опціонально) ~$10/рік
Загалом за курс ~$25-45

Видаліть дроплет після курсу, щоб зупинити нарахування. Нові акаунти DigitalOcean отримують $200 безкоштовного кредиту — достатньо, щоб пройти курс безкоштовно.
